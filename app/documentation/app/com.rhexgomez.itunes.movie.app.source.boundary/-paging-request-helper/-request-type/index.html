Available request types.

Enum Values

Corresponds to an initial request made to a DataSource or the empty state for a BoundaryCallback.

Corresponds to the loadBefore calls in DataSource or onItemAtFrontLoaded in BoundaryCallback.

Corresponds to the loadAfter calls in DataSource or onItemAtEndLoaded in BoundaryCallback.
