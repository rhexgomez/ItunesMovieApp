runIfNotRunning

@AnyThread fun runIfNotRunning(type: RequestType, request: Request): Boolean

Runs the given Request if no other requests in the given request type is already running.

If run, the request will be run in the current thread.

Parameters

type - The type of the request.

request - The request to run.

Return
True if the request is run, false otherwise.
