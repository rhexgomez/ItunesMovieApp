Request

typealias Request = (Callback) -> Unit

Runner class that runs a request tracked by the PagingRequestHelper.

When a request is invoked, it must call one of Callback.recordFailure or Callback.recordSuccess once and only once. This call can be made any time. Until that method call is made, PagingRequestHelper will consider the request is running.
